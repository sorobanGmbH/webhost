Unsere Apps verarbeiten keine persönlichen Daten. Alle Funktionen laufen lokal auf Ihrem Gerät. Es findet keine Übertragung von Daten auf unsere Server statt.

Wir verwenden keine Analytiksoftware oder Tracking-Tools. Es findet keine Sammlung oder Auswertung von Nutzungsdaten statt.

Da keine Daten übertragen oder langfristig gespeichert werden, entstehen keine Risiken durch externe Zugriff von unserer Seite.

Wir behalten uns das Recht vor, diese Datenschutzbestimmungen zu aktualisieren. Änderungen werden in der App oder auf unserer Website bekanntgegeben.
